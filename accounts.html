Import advertisers

Paste rows from a Google Sheet (or any CSV) into the relevant tab below. Only Advertisers is required — Articles are optional and link to their advertiser by name. Need additional contacts beyond the primary? Add them per-advertiser in the wizard after import.

Fill in each sheet, save, upload — auto-routes to tabs
Recognised columns
name (or company / advertiser) — required
status — Active / In Negotiation / Paused / Churned
term_type (or term) — Monthly Rolling / Quarterly Rolling / Fixed
par_live (or par) — Yes / No
start_date, renewal_date — YYYY-MM-DD or DD/MM/YYYY
fixed_months — number, only for Fixed term
first_name / last_name / email — primary contact (one per advertiser)
Recognised columns
advertiser — required, must match an advertiser name
url (or link) — full article URL
position (or pos) — 1 to 10
monthly_price (or price) — what the advertiser pays for this article (only for standalone articles)
bundle — tag to group articles into a package. Same tag (per advertiser) = same bundle. Use any value (1, 2, A, B…). Leave empty for standalone.
bundle_price — combined monthly fee for the bundle. Fill this on every row of the bundle (just copy down).
active — TRUE / FALSE / Yes / No (default TRUE)
Bundle tags are scoped per advertiser — you can reuse 1, 2 across different advertisers without merging them. Bundles get auto-named "Package 1", "Package 2" — rename later in the wizard if you like.
